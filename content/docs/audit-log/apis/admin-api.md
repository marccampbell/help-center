## Authentication

The Admin API expects a base64-ecoded hmac-signed [JSON Web Token](https://tools.ietf.org/html/rfc7519) in an `Authorization` header:

```
Authorization: WU9VUl9TSUdORURfSldUX1RPS0VOX18K...
```
